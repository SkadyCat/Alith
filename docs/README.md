# 欢迎使用文档服务

这是一个基于 **Markdown** 的文档服务系统。

## 功能特性

- 📁 左侧文件树导航
- ✏️ 左编辑右预览模式
- 🎨 优雅的界面设计
- 💾 实时保存文档

## 快速开始

1. 在左侧菜单选择或创建文档
2. 点击顶部 **编辑** 按钮进入编辑模式
3. 在左侧编写 Markdown，右侧实时预览
4. 点击 **保存** 保存你的文档
