## 根本原因

**不是 POLL 慢，是 Claude API 调用链慢。**

每次工具调用（`read_powershell`、`grep`、`view`...）都要：
1. 将**完整上下文**（系统设定 + 会话历史 + 工具结果）发送给 Claude API
2. 等 Claude 推理 → 返回下一步
3. 重复

上下文越大 → 每次往返越慢。当前会话历史已很长，所以每轮 10-20 秒。
