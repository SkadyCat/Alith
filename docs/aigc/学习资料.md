## 快速上手小贴士
- 模型路径（Portable 版常见）：  
  ComfyUI/models/checkpoints/ （Checkpoint/Flux 主模型）  
  ComfyUI/models/loras/ （LoRA）  
  ComfyUI/models/vae/ （VAE）  
- 安装 ComfyUI-Manager 后 → Manager → Model Manager → 直接搜 Civitai 模型下载。  
- 工作流示例：Civitai 搜 "comfyui workflow" 下载 JSON 拖入测试。  
  https://civitai.com/search/models?query=comfyui%20workflow&sortBy=models_v9

这些链接覆盖从零基础到高级用法。如果你需要特定模型（如 Flux）的路径或工作流示例，再告诉我！
