# 腾讯混元 Hunyuan3D 核心学习文档

> 更新于 2026年3月 | 基于最新 Hunyuan3D-2 / 2.1 版本

---

## 一、项目概述

**腾讯混元 Hunyuan3D** 是腾讯自研的高精度3D内容生成系统，包含两大核心组件：

| 组件 | 说明 |
|------|------|
| **Hunyuan3D-DiT** | 基于Flow-based扩散Transformer的形状生成模型 |
| **Hunyuan3D-Paint** | 高分辨率纹理合成模型（支持PBR材质） |

### 版本历史
| 版本 | 发布时间 | 亮点 |
|------|---------|------|
| 2.0 | 2025-01 | 开源推理代码+预训练权重，支持文/图生3D |
| 2.0-Turbo | 2025-03 | 加速版，推理速度提升2倍 |
| 2.1 | 2025-06 | 全开源（含训练代码），新增PBR模型+VAE编码器 |
| 2.5 | 2025-06 | 系统级技术报告 |

---

## 二、核心能力

- **文生3D**：中文提示词 → 高精度带纹理3D模型
- **图生3D**：单张或多视角图片 → 3D重建
- **PBR材质**：物理渲染材质（Base Color / Metallic / Roughness / Normal）
- **多格式输出**：GLB、OBJ、STL、FBX、USDZ
- **LowPoly模式**：智能拓扑，低面数（3000面起）
- **Sketch模式**：草图/线稿转3D

## 三、腾讯云 API 快速上手

### 接口信息
- **域名**：`ai3d.tencentcloudapi.com`
- **版本**：`2025-05-13`
- **服务名**：`ai3d`
- **鉴权**：TC3-HMAC-SHA256

### 主要接口

| Action | 说明 |
|--------|------|
| `SubmitHunyuanTo3DProJob` | 提交专业版任务（文/图生3D） |
| `QueryHunyuanTo3DProJob` | 查询专业版任务结果 |
| `SubmitHunyuanTo3DRapidJob` | 提交极速版任务 |
| `QueryHunyuanTo3DRapidJob` | 查询极速版任务结果 |
| `SubmitTextureTo3DJob` | 提交纹理生成任务 |
| `Convert3DFormat` | 模型格式转换 |

### 提交任务示例（文生3D）
```json
POST https://ai3d.tencentcloudapi.com
X-TC-Action: SubmitHunyuanTo3DProJob
X-TC-Version: 2025-05-13

{
  "Prompt": "一只可爱的青蛙，Q版卡通风格，绿色皮肤，大眼睛",
  "Model": "3.0",
  "GenerateType": "Normal",
  "FaceCount": 500000
}
```

### 查询结果
```json
{
  "Response": {
    "Status": "DONE",
    "ResultFile3Ds": [
      {
        "Type": "GLB",
        "Url": "https://cos.ap-guangzhou.tencentcos.cn/xxx.glb",
        "PreviewImageUrl": "https://cos.ap-guangzhou.tencentcos.cn/xxx.png"
      }
    ]
  }
}
```

### 任务状态
| Status | 含义 |
|--------|------|
| `WAIT` | 等待中 |
| `RUN` | 生成中 |
| `DONE` | 成功 |
| `FAIL` | 失败 |

## 四、本地部署（ComfyUI）

```bash
# 1. 安装 ComfyUI
git clone https://github.com/comfyanonymous/ComfyUI
cd ComfyUI && pip install -r requirements.txt

# 2. 安装 Hunyuan3D 节点
cd custom_nodes
git clone https://github.com/kijai/ComfyUI-Hunyuan3DWrapper
cd ComfyUI-Hunyuan3DWrapper && pip install -r requirements.txt

# 3. 下载模型权重（Hugging Face）
# tencent/Hunyuan3D-2 或 tencent/Hunyuan3D-2.1

# 4. 启动
cd ../../ && python main.py --port 8188
```

## 五、混元应用（本地服务）

本应用位于 `application/hunyuan/`，端口 **8044**。

### 功能
- `POST /api/generate` — 提交文生3D任务（需腾讯云账号开通服务）
- `GET  /api/job/:jobId` — 查询任务状态
- `POST /api/job/:jobId/upload` — 下载结果并上传COS
- `GET  /api/models` — 列出已生成模型

### 快速生成青蛙模型
```bash
# 程序化生成（无需API）
node E:\docs-service\application\hunyuan\create_frog.py

# 上传到COS
node E:\docs-service\application\hunyuan\upload-to-cos.js
```

### 青蛙模型 COS 地址
```
https://magicworld-1304036735.cos.ap-guangzhou.myqcloud.com/hunyuan3d/frog_model.glb
```

## 六、API 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `Prompt` | String | 中文描述（最多1024字符） |
| `Model` | String | `3.0`（默认）或 `3.1` |
| `GenerateType` | String | Normal / LowPoly / Geometry / Sketch |
| `FaceCount` | Integer | 面数，默认50万，范围1万~150万 |
| `EnablePBR` | Boolean | 是否开启PBR材质（默认false） |
| `ResultFormat` | String | 额外格式：STL / USDZ / FBX |
| `ImageUrl` | String | 图生3D的图片URL（与Prompt二选一） |
| `ImageBase64` | String | 图生3D的Base64图片 |

## 八、注意事项

1. **服务开通**：腾讯云 AI3D API 需要在控制台手动开通并充值（按量计费）
2. **并发限制**：默认3个并发任务，超出会排队
3. **生成时间**：专业版通常需要3~10分钟
4. **许可证**：Tencent Hunyuan Community License，支持商用但有限制
5. **格式推荐**：GLB格式兼容性最好（Three.js/模型查看器均支持）
