# Canvas Editor 应用文档

> **应用路径**: `E:\docs-service\application\canvas-editor`  
> **运行端口**: `8331`  
> **访问地址**: `http://localhost:8331`  
> **启动命令**: `node server.js`

---

## 概述

Canvas Editor 是一款基于 HTML5 Canvas + DOM 的可视化界面布局编辑器，运行在本地 Node.js 服务中。  
用户可在网格画布上绘制、移动、调整矩形框（"Box"），用于快速构建界面原型或区域示意图。

同时内置了文档查看器，连接 `http://localhost:7439` 的文档服务，可在编辑器中直接浏览系统文档。

## 核心功能

### 1. 编辑模式

| 模式 | 快捷键 | 说明                                           |
|------|--------|------------------------------------------------|
| 选择 | `V`    | 点击选中元素，拖拽移动，8 向手柄缩放           |
| 绘制 | `B`    | 在画布上拖拽创建新矩形，完成后自动切回选择模式 |

### 2. 快捷键

| 快捷键               | 功能                              |
|----------------------|-----------------------------------|
| `V`                  | 切换到选择模式                    |
| `B`                  | 切换到绘制模式                    |
| `Del` / `Backspace`  | 删除选中元素                      |
| `Escape`             | 取消选中 / 关闭文档查看器         |
| `Ctrl+Z`             | 撤销（最多 50 步）                |
| `Ctrl+Y`             | 重做                              |
| `← ↑ ↓ →`           | 微移选中元素（1px）               |
| `Shift + 方向键`     | 大幅移动选中元素（10px）          |
| `Ctrl + 滚轮`        | 缩放画布（范围：25% ~ 300%）      |

### 3. 元素（Box）属性

每个 Box 是一个矩形 DOM 元素，包含以下可编辑属性：

| 属性          | 类型    | 说明                             |
|---------------|---------|----------------------------------|
| `id`          | Number  | 自增唯一标识                     |
| `x`, `y`      | Number  | 左上角坐标（对齐到吸附网格）     |
| `w`, `h`      | Number  | 宽高（最小值均为 20px）          |
| `label`       | String  | 显示名称（默认 Box1, Box2…）    |
| `borderColor` | String  | 边框颜色（默认 `#7c6af7`，紫色） |
| `bgColor`     | String  | 背景填充色（默认半透明紫）       |
| `borderWidth` | Number  | 边框粗细（1 ~ 10px）             |
| `opacity`     | Number  | 整体透明度（0.1 ~ 1.0）          |

### 4. 网格与对齐

- **网格显示**：小格 20px（`#2a2a40`），大格 100px（`#303050`）
- **吸附步长**：10px，移动/缩放时自动吸附
- 顶部 `网格` 和 `对齐` 复选框可独立开关

### 5. 预设形状

点击左侧面板预设项，在画布中心快速插入：

| 名称   | 尺寸（W × H） |
|--------|---------------|
| 矩形   | 120 × 80      |
| 大框   | 200 × 120     |
| 正方形 | 80 × 80       |
| 横条   | 160 × 40      |
| 竖条   | 40 × 160      |

### 6. 图层面板

- 显示画布中所有 Box（逆序，最新在顶部）
- 点击任意图层条目可选中对应元素

### 7. 属性面板（右侧）

选中元素后显示实时可编辑属性：
- **位置**：X、Y 数字输入框
- **尺寸**：W、H 数字输入框
- **名称**：文字输入
- **边框**：宽度（数字） + 颜色选择器
- **透明度**：拖动滑块（0.1 ~ 1.0），实时百分比显示

### 8. 撤销 / 重做

- 历史记录栈最大 50 步
- 每次移动、缩放、删除、清空操作前自动保存快照
- 工具栏按钮或 `Ctrl+Z` / `Ctrl+Y` 触发

### 9. 画布缩放

- 范围：**25%** ~ **300%**
- 工具栏 `+` / `−` 每次 ±10%；`重置` 还原到 100%
- `Ctrl + 滚轮` 快速缩放
- 网格始终以视口尺寸绘制，不受 zoom 影响

### 10. 区域识别（Zone Highlight）

工具栏 **识别** 按钮开启 DOM 元素拾取模式：

1. 悬停：浮层蓝色高亮 + 标签提示（tag / id / class）
2. 点击：底部弹出元素详情面板（含 data 属性）
3. 一键 **复制** 元素信息到剪贴板
4. `Esc` 退出

此功能可用于快速识别界面中任意 DOM 元素的位置和属性。

### 11. 内置文档面板

左侧底部文档区域，通过 `http://localhost:7439/open/proxy` 访问系统文档：

| 功能       | 说明                                    |
|------------|-----------------------------------------|
| 树形结构   | 展示 `docs/` 目录，支持文件夹折叠       |
| 实时搜索   | 按文件名过滤，关键词高亮显示            |
| 文档查看   | 点击文件在浮层中渲染 Markdown HTML      |
| 新建文件夹 | 输入相对路径，在 `docs/` 下创建目录     |
| 折叠/展开  | 可折叠文档面板，节省画布空间            |

### 12. 底部控制台

记录所有操作日志，支持 5 种日志类型：

| 类型    | 颜色 | 示例                              |
|---------|------|-----------------------------------|
| `info`  | 白色 | 切换模式、初始化                  |
| `ok`    | 绿色 | 绘制/添加元素成功                 |
| `warn`  | 橙色 | 删除元素、清空画布                |
| `error` | 红色 | 文档加载失败、网络错误            |
| `dim`   | 灰色 | 撤销、移动、打开文档等次要操作    |

## 数据结构

```javascript
// 内存中的 boxes 数组
boxes = [
  {
    id: 1,
    x: 100, y: 80,
    w: 200, h: 120,
    label: 'Box1',
    borderColor: '#7c6af7',
    bgColor: 'rgba(124,106,247,0.06)',
    borderWidth: 2,
    opacity: 1.0
  },
  // ...
]
```

> **注意**：数据仅保存在内存中，刷新页面后丢失。

## 已知限制

| 限制             | 说明                                  |
|------------------|---------------------------------------|
| 无持久化         | 刷新页面后所有元素丢失，暂无保存功能  |
| 仅矩形元素       | 不支持椭圆、文字框、连线等形状        |
| 无元素复制       | 暂无 Ctrl+C / Ctrl+V 功能             |
| 单用户本地       | 不支持多人协作                        |
| 文档面板依赖服务 | 需要 `http://localhost:7439` 在线     |

## 扩展建议

- **数据持久化**：JSON 导入 / 导出，或保存到 `localStorage`
- **元素复制**：`Ctrl+C` / `Ctrl+V` 支持
- **更多形状**：椭圆、文字框、箭头连线
- **导出图片**：将画布渲染为 PNG 下载
- **多页面支持**：管理多张画布
- **右键菜单**：快速访问常用操作
