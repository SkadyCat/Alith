# 推荐的免费开源AI Agent框架 - 全部官方链接汇总 (2026年3月)

1. LangChain (包括LangGraph扩展)
   - 官方文档: https://docs.langchain.com/
   - LangGraph 文档 (推荐用于复杂Agent): https://docs.langchain.com/ (搜索 LangGraph 或直接访问子路径)
   - GitHub 主仓库 (LangChain): https://github.com/langchain-ai/langchain
   - GitHub LangGraph: https://github.com/langchain-ai/langgraph

2. CrewAI
   - 官方文档: https://docs.crewai.com/
   - GitHub 主仓库: https://github.com/crewAIInc/crewAI
   - 示例仓库 (完整应用示例): https://github.com/crewAIInc/crewAI-examples

3. AutoGen (Microsoft)
   - 官方文档: https://microsoft.github.io/autogen/ (或 https://microsoft.github.io/autogen/stable/index.html)
   - GitHub 主仓库: https://github.com/microsoft/autogen

4. OpenAI Agents SDK
   - 官方文档: https://openai.github.io/openai-agents-python/
   - GitHub Python版: https://github.com/openai/openai-agents-python
   - GitHub JavaScript/TypeScript版 (可选): https://github.com/openai/openai-agents-js
   - OpenAI 平台相关Agents指南: https://developers.openai.com/api/docs/guides/agents-sdk  (或 https://platform.openai.com/docs/guides/agents)
