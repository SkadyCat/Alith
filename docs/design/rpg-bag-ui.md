# RPG 背包 UI 设计规范

> 研究来源：Diablo II、Path of Exile、暗黑破坏神 III、深岩银河等经典 ARPG
> 适用对象：canvas-editor bag2.session

---

## 经典 RPG 背包的三大区域

```
┌──────────────────────────────────────────────────────────────┐
│  TITLE BAR：游戏标题感重要，带图标+金属质感边框                │
├────────────────┬──────────────────────────┬──────────────────┤
│  角色属性面板  │     装备纸娃娃面板        │   物品栏网格     │
│  (左侧, 约20%) │     (中间, 约30%)        │   (右侧, 约50%)  │
│                │                          │                  │
│  • STR/DEX/    │  • 装备槽围绕人体轮廓    │  • 等方格网格    │
│    INT/VIT     │  • HEAD 顶部居中         │  • 每格60-64px   │
│  • HP/MP条     │  • CHEST 中央            │  • 深色内阴影    │
│  • 负重条      │  • LHAND/RHAND 两侧      │  • 悬停高亮      │
│  • 金币显示    │  • LEGS 下方             │  • 数量角标      │
├────────────────┴──────────────────────────┴──────────────────┤
│  快速栏 Belt：4-8个快捷格，通常放药水/消耗品                   │
└──────────────────────────────────────────────────────────────┘
```

## 当前 bag2.session 布局（960×620）

| 区域 | x | y | w | h | 说明 |
|------|---|---|---|---|------|
| TitleBar | 0 | 0 | 960 | 36 | 深色羊皮纸风格标题栏 |
| StatPanel | 0 | 36 | 200 | 534 | 角色属性（STR/DEX/INT/VIT + HP/MP/负重/金币）|
| EquipPanel | 200 | 36 | 280 | 534 | 12个装备槽（头/颈/武/盾/胸/手/腰/腿/靴/戒×2/护符）|
| InvPanel | 480 | 36 | 480 | 534 | 7×7 = 49个物品格 |
| BeltPanel | 0 | 570 | 960 | 50 | 8个快捷格 + 金币显示 |

---

## 为什么之前的3横栏布局不对？

| 问题 | 原因 |
|------|------|
| 没有纸娃娃（Paperdoll）| RPG 装备的核心交互是"穿在角色身上看效果" |
| 等宽3列像设置面板 | 游戏UI需要视觉重心，物品栏应该最大 |
| 缺少快捷栏 | Belt/快捷槽是RPG操作必需品 |
| 没有深色质感背景 | 现代游戏背包都有皮革/木纹/羊皮纸等材质感 |

## 经典装备槽位置规律（Paperdoll）

```
        [头部]  [颈部]
[武器]  [胸甲]  [盾牌]
[手套]  [腰带]  [戒指]
[戒指]  [腿甲]  [护符]
        [靴子]
```

装备槽颜色：空=暗金边框 `#5a4020`，悬停=亮金 `#d4a84b`，已装=发光轮廓

## 颜色规范（暗色奇幻风）

| 用途 | 色值 |
|------|------|
| 画布背景 | `#0a0703` 极深棕黑 |
| 面板背景 | `#110c04` ~ `#1a1008` 深棕 |
| 边框 | `#4a3010` 暗金棕 |
| 标题/金色文字 | `#d4a84b` 暗金 |
| 普通文字 | `#c8b48a` 羊皮纸色 |
| HP条 | fill=`#c0392b` bar=`#2c1010` |
| MP条 | fill=`#2980b9` bar=`#0d1020` |
| 负重条 | fill=`#8b6914` bar=`#1a1408` |
| STR | `#e05050` | DEX | `#50c050` | INT | `#5090e0` | VIT | `#e0a030` |

## 物品槽网格计算

```
格子尺寸: 62×62px
间距: 4px
步长: 66px
7列: 7×66-4 = 458px (适配 480px 面板)
7行: 7×66-4 = 458px (适配 534px 面板)
```
