# 使用指南

## 基础操作

### 创建文档
点击左侧菜单顶部的 **+** 按钮，输入文件名即可创建新文档。

### 编辑文档
选中文档后，点击顶部工具栏的 **编辑** 按钮切换到编辑模式。

### 保存文档
编辑完成后，点击 **保存** 按钮或使用快捷键 `Ctrl+S` 保存。

## Markdown 语法

### 标题
```
# H1 标题
## H2 标题
### H3 标题
```

### 文本格式
- **加粗**: `**文字**`
- *斜体*: `*文字*`
- ~~删除线~~: `~~文字~~`
- `行内代码`: ``代码``

### 列表
```
- 无序列表项
- 另一项
  - 嵌套项

1. 有序列表项
2. 另一项
```

### 链接和图片
```
[链接文字](https://example.com)
![图片描述](image.png)
```
