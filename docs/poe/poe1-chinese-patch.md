## 已安装补丁

使用 **PoeChinese3 v2.7.3** (by aianlinb) 成功应用了**繁体中文 (Traditional Chinese)** 汉化补丁。

- 工具路径：G:\poegj\PoeChinese3_win-x64.exe
- 修改时间：2026-03-07 01:10:31
- 功能：启用繁體中文語系（流亡黯道 Traditional Chinese）

## 如何切换语言

1. 启动 POE1（G:\poegj\PathOfExile.exe）
2. 进入 **Options（选项）→ UI → Language**
3. 选择 **繁體中文** 即可

## 简体中文（进阶）

如需简体中文，可使用 VisualGGPK2 手动替换 languages.dat：

1. 打开 G:\poegj\游戏汉化工具\VisualGGPK2_New\VisualGGPK2.exe
2. 打开 G:\poegj\Content.ggpk
3. 导航到 ROOT > Bundles2 > data > languages.dat
4. 替换为 G:\poegj\游戏汉化补丁\ROOT\Bundles2\data\languages.dat

## 注意

- 每次游戏大版本更新后，Content.ggpk 会被覆盖，需重新应用补丁
- 繁体中文显示效果完整（GGG 官方台湾区翻译）
- 简体中文可能文字不完整（部分字体需要单独替换）
