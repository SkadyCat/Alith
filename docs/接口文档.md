# 对外接口文档

本服务通过 `/open` 前缀暴露对外开放接口，供内部系统/自动化工具直接向 DocSpace 提交、搜索文档。  
通过 `/tools` 前缀提供代码执行工具，供 Agent 调用 Python 或 Shell 脚本。

| 接口 | 方法 | 说明 |
|---|---|---|
| `/open/submit` | POST | 新建或覆盖文档 |
| `/open/search` | POST | 全文模糊搜索 |
| `/open/study`  | GET  | 获取使用说明文档 |
| `/tools/python` | POST | 执行 Python 代码片段 |
| `/tools/shell`  | POST | 执行 PowerShell 命令/脚本 |

---

## POST `/open/submit` — 提交文档

新建或覆盖一篇文档。

### 请求

**Content-Type**: `application/json`

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|---|---|:---:|:---:|---|
| `filename` | string | ✅ | — | 文件名或相对路径，如 `guide`、`api/intro`，可省略 `.md` 后缀 |
| `content` | string | ✅ | — | Markdown 正文内容 |
| `filetype` | string | — | `md` | 文件类型，当前仅支持 `md` |
| `overwrite` | boolean | — | `true` | `true` 时覆盖同名文档，`false` 时若文件已存在则返回 409 |

### 响应

**成功 200**

```json
{
  "success": true,
  "path": "api/intro.md",
  "overwritten": false,
  "message": "文档已创建",
  "html": "<h1>...</h1>"
}
```

**文件已存在且 overwrite: false → 409**

```json
{
  "success": false,
  "error": "文档已存在，如需覆盖请设置 overwrite: true",
  "path": "api/intro.md"
}
```

**参数错误 → 400**

```json
{
  "success": false,
  "error": "filename 为必填项"
}
```

---

## 示例

### cURL

```bash
# 新建文档
curl -X POST http://localhost:7439/open/submit \
  -H "Content-Type: application/json" \
  -d '{
    "filename": "api/intro",
    "content": "# 简介\n\n这是通过接口提交的文档。",
    "overwrite": true
  }'
```

### Python

```python
import requests

resp = requests.post("http://localhost:7439/open/submit", json={
    "filename": "api/intro",
    "content": "# 简介\n\n这是通过接口提交的文档。",
    "overwrite": True
})
print(resp.json())
```

### Node.js / fetch

```javascript
const res = await fetch("http://localhost:7439/open/submit", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({
    filename: "api/intro",
    content: "# 简介\n\n这是通过接口提交的文档。",
    overwrite: true
  })
});
const data = await res.json();
console.log(data);
```

## 路径规则

- `filename` 支持子目录，如 `notes/daily` → 保存为 `docs/notes/daily.md`
- 目录不存在时自动创建
- 路径不允许包含 `..`（防目录穿越），否则返回 403

## POST `/open/search` — 全文模糊搜索

在所有文档的标题和正文中搜索，支持四种匹配模式。

### 请求（JSON）

| 字段 | 类型 | 必填 | 默认 | 说明 |
|---|---|:---:|:---:|---|
| `q` | string | ✅ | — | 搜索词 |
| `mode` | string | — | `fuzzy` | 匹配模式（见下表） |
| `fields` | string | — | `both` | 搜索范围：`title` / `content` / `both` |
| `limit` | number | — | `20` | 最多返回条数（上限 100） |

### 匹配模式

| mode | 说明 |
|---|---|
| `fuzzy` | 字符子序列匹配（类 fzf），字符依次出现即命中 |
| `words` | 多词全匹配，空格分隔，所有词均需出现 |
| `exact` | 精确子串匹配（忽略大小写） |
| `regex` | 正则表达式（多行、忽略大小写） |

### 响应

```json
{
  "success": true,
  "query": "接口",
  "mode": "fuzzy",
  "fields": "both",
  "total": 2,
  "results": [
    {
      "path": "接口文档.md",
      "title": "对外接口文档",
      "score": 15,
      "excerpts": ["本服务通过 /open 前缀暴露对外开放接口..."]
    }
  ]
}
```

### 示例

```bash
# 模糊搜索（默认）
curl -X POST http://localhost:7439/open/search \
  -H "Content-Type: application/json" \
  -d '{"q":"接口文档"}'

# 多词匹配，只搜标题
curl -X POST http://localhost:7439/open/search \
  -H "Content-Type: application/json" \
  -d '{"q":"使用 指南","mode":"words","fields":"title"}'

# 正则搜索
curl -X POST http://localhost:7439/open/search \
  -H "Content-Type: application/json" \
  -d '{"q":"^## .*(接口|API)","mode":"regex","limit":5}'
```

## GET `/open/study` — 获取使用说明

返回《如何使用文档》的 Markdown 原文与 HTML，供外部系统直接嵌入展示。

```bash
curl http://localhost:7439/open/study
```

### 响应

```json
{
  "success": true,
  "path": "如何使用文档.md",
  "content": "# 如何使用 DocSpace\n...",
  "html": "<h1>如何使用 DocSpace</h1>..."
```

# 工具执行接口 (`/tools`)

> **适用场景**：供 Agent 在自动化任务中执行代码或系统命令，结果以结构化 JSON 返回。  
> **安全说明**：接口不对外暴露，请仅在本机或受信内网环境中使用。

工具脚本位于 `tools/` 目录：

| 文件 | 说明 |
|---|---|
| `tools/python_runner.py` | Python 沙箱执行器，预置常用标准库 |
| `tools/shell_runner.ps1` | PowerShell 命令执行器 |
| `tools/requirements.txt` | Python 可选依赖包清单 |

---

## POST `/tools/python` — 执行 Python 代码

执行任意 Python 代码片段，捕获标准输出/错误并返回。

**预置可用模块**：`os`, `sys`, `re`, `math`, `time`, `datetime`, `json`, `hashlib`, `base64`, `urllib`, `collections`, `itertools`, `requests`（需安装）

### 请求

**Content-Type**: `application/json`

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|---|---|:---:|:---:|---|
| `code` | string | ✅ | — | Python 代码字符串 |
| `timeout` | number | — | `30000` | 超时毫秒，最大 `120000` |

### 响应

**成功 200**

```json
{
  "success": true,
  "stdout": "hello world\n",
  "stderr": "",
  "exitCode": 0
}
```

**运行时错误**

```json
{
  "success": false,
  "stdout": "",
  "stderr": "Traceback (most recent call last):\n  ...\nZeroDivisionError: division by zero",
  "exitCode": 1
}
```

### 示例

```bash
# 基础计算
curl -X POST http://localhost:7439/tools/python \
  -H "Content-Type: application/json" \
  -d '{"code": "print(sum(range(1, 101)))"}'

# 多行代码
curl -X POST http://localhost:7439/tools/python \
  -H "Content-Type: application/json" \
  -d '{
    "code": "import datetime\nprint(datetime.date.today())",
    "timeout": 5000
  }'
```

## POST `/tools/shell` — 执行 PowerShell 命令

执行 PowerShell 命令或多行脚本，捕获输出并返回。

### 请求

**Content-Type**: `application/json`

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|---|---|:---:|:---:|---|
| `command` | string | ✅ | — | PowerShell 命令或多行脚本 |
| `timeout` | number | — | `30000` | 超时毫秒，最大 `120000` |

### 响应

**成功 200**

```json
{
  "success": true,
  "stdout": "C:\\docs-service\\tools",
  "stderr": "",
  "exitCode": 0
}
```

**命令错误**

```json
{
  "success": false,
  "stdout": "",
  "stderr": "Get-ChildItem : Cannot find path ...",
  "exitCode": 1
}
```

### 示例

```bash
# 查看当前目录
curl -X POST http://localhost:7439/tools/shell \
  -H "Content-Type: application/json" \
  -d '{"command": "Get-Location"}'

# 列出 docs 目录
curl -X POST http://localhost:7439/tools/shell \
  -H "Content-Type: application/json" \
  -d '{"command": "Get-ChildItem ..\docs"}'

# 多行脚本
curl -X POST http://localhost:7439/tools/shell \
  -H "Content-Type: application/json" \
  -d '{
    "command": "$date = Get-Date\n$date.ToString(\"yyyy-MM-dd HH:mm:ss\")",
    "timeout": 10000
  }'
```
